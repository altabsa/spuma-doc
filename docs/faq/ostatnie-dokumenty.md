# Gdzie znajdę dokument, który ostatnio oglądałem?  

**Zgłoszony problem**  
Użytkownik chce szybko wrócić do dokumentu, który niedawno otwierał.  

**Rozwiązanie**  
Przejdź do zakładki **Ostatnie**.  

SPUMA automatycznie zapisuje listę otwieranych dokumentów.
